initmain

Entry point for Noona-Vault. Initializes databases, configures Express middleware, mounts REST API routes, and handles lifecycle events including graceful shutdown. This script is designed to be containerized (Docker) and runs the Vault microservice which handles secure storage, JWT distribution, and database operations.
